---------------------------
---	Preparing product	---
---------------------------

Use 'scripts/deploy.cmd' to create executable jar file, resources and run.cmd script.
The above files will appear in 'target' folder.

Use 'target/run.cmd' script to run application.

-----------------------
---	Other scripts	---
-----------------------

Use 'scripts/clean.cmd' to clean 'target' folder.

Use 'scripts/run.cmd' to launch application that is not using jar file.
